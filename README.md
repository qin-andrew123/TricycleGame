# TricycleGame
Game Idea from gamejam that I wanted to make

# Premise
- Inspired from the Japanese TV show Old Enough!
- Odyssey - where it takes Odysseus forever to make it back home
- General idea is that a kid goes to get a carton of eggs from grocery store and must make it home
- However, their tricycle is broken and they can only spin CW/CCW

# Genre
- Roguelike/Randomly Generated levels
- Puzzle game akin to frogger
- Getting over it, games with frustrating controls

# Controls
- WASD/Arrows | XYAB/Joypad Arrows/Sticks - Moving the tricycle

# Perspective
- Top Down, potentially 3/4ths view?

# Visual Style
- Heavily Abstracted. Think maybe like blood dungeon?
- Want it to be from POV of the kid, absurdist like Oyasumi PunPun? Without the excess edginess

# World Gen
- World is built like a grid of streets, the store starts in one corner and the house is in the other
- You select which street you want to go on, you cannot backtrack into a different lane'
  - Streets are further divided into chunks/tiles
 
# Obstacles
- Cars
- Birds
- Dogs

Each does a unique behavior and is something that the player must navigate

# Mechanics
- Stop the tricycle
- Move faster
- Change direction
- throw egg (potentially make it so that birds can replenish eggs in carton?) lives

# Dialogue System
- Lets explore the lore a little more
